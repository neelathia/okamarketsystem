

ROLE: Senior Documentation Manager
COMPANY: RevenueCat
DEPARTMENT: Documentation (cross-functional: Product, Engineering, Support, Marketing)
LOCATION: Remote (global)
SALARY: $169,000 USD
EMPLOYMENT TYPE: Full-time

---

STRUCTURE

RESPONSIBILITIES:
- Own documentation strategy and information architecture across the product
- Define structure supporting new apps, migrations, and feature-subset users
- Establish consistency in voice, style, and quality
- Partner with Engineering and Product to ship documentation alongside features
- Make documentation useful for both humans and AI (LLMs, search, AI-driven tooling)
- Expand documentation into a broader learning ecosystem (video, blog, tutorials)
- Design and implement systems to keep documentation current over time
- Establish documentation as source of truth — internal and external

REQUIREMENTS:
- 5–8+ years in technical documentation, developer education, or technical content leadership
- Ownership of documentation for complex, evolving developer platform or API-driven product
- Hands-on with git-based, markdown-driven documentation systems (Docusaurus a plus)
- Strong editorial mindset with enough technical depth to work with engineers
- Track record of designing processes that keep documentation accurate over time

NICE TO HAVE:
- Docusaurus experience

---

FUNCTION (OKA INTENT)

DO:
Design a documentation system that scales with the product, serves both human readers and AI systems, and functions as a durable competitive advantage.

FIX:
Strong individual documentation pages that don't form a cohesive system. Documentation lags features, varies in tone, and mixes content types in ways that create customer friction.

CHECK:
Documentation coherence, feature coverage, AI readability, customer orientation speed, support deflection, internal onboarding use.

CONSTRAINT:
First hire to own end-to-end documentation experience. Must build the system without a pre-existing playbook.

---

INTERPRETATION

CORE IDEA:
RevenueCat is treating documentation as a product — one with strategy, architecture, metrics, and a roadmap. The Senior Documentation Manager is the product owner of that system.

IMPLIED NEED:
They have content without a system. The role is explicitly chartered to build the system, not produce more content.

MATURITY LEVEL:
Architecture

RELATED ROLES:
- Principal Technical Writer / Crunchafi (documentation as strategic capability)
- LLM-Aware Technical Editor (AI readability as first-class concern)
- Senior Director Content Strategy / Fueled (content as scalable system)

PATTERN:
Developer-facing companies are recognizing that documentation is a retention and adoption lever — and they are investing in it at the architecture level, not the production level.

---

YOUR POSITION:
RevenueCat is not hiring a documentation manager. They are hiring the person who will decide what documentation is, how it works, and what it produces — for humans and machines simultaneously. At $169K, this is being priced as a product leadership role. The market is catching up to what documentation actually does.
